## 🆘 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ:

```bash
ssh root@your-digital-ocean-ip

```

**Telegram боты не отправляют:**
```
1. Проверь токены в .env на сервере
2. Проверь Chat ID в test-telegram-bots.ts
3. Запусти тест локально: npx ts-node test-telegram-bots.ts
```

**База данных не работает:**
```
1. Supabase Dashboard → Settings → API
2. Проверь API keys в .env на сервере
3. Проверь Supabase URL
```
