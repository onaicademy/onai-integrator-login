## 📊 ИЕРАРХИЯ Z-INDEX (СНИЗУ ВВЕРХ):

```
z-10     → Фоновые декоративные элементы (звезды, линии)
z-50     → Обычные UI элементы
z-[100]  → Обертки кнопок Admin
z-101    → Сами кнопки Admin (inline style)
z-[9999] → DialogOverlay (темный фон)
z-[10000]→ DialogContent (само окно)
z-10001  → LessonEditDialog & ModuleEditDialog (inline style - МАКСИМАЛЬНЫЙ!)
```

## 🚨 ЕСЛИ КНОПКИ ВСЁ РАВНО НЕ РАБОТАЮТ:

### Проблема 1: НЕТ ЛОГОВ В КОНСОЛИ

**Решение:**
1. Откройте F12 → Console
2. Проверьте фильтр (должен быть ALL)
3. Перезагрузите страницу `Ctrl+Shift+R`
4. Проверьте есть ли:
   ```
   🚨 MODULE.TSX ЗАГРУЗИЛСЯ!
   ```

Если НЕТ → компонент вообще не рендерится!

### Проблема 2: КНОПКИ НЕ ВИДНЫ

**Проверьте:**
```javascript
// В консоли:
console.log('isAdmin:', isAdmin);
```

Если `isAdmin: false` → вы НЕ админ!

**Решение:**
```sql
-- В Supabase SQL Editor:
UPDATE users 
SET role = 'admin' 
WHERE email = 'ваш@email.com';
```

Затем выйдите и войдите заново.

### Проблема 3: КНОПКИ ВИДНЫ, НО НЕ КЛИКАБЕЛЬНЫ

**Проверьте в DevTools:**
1. F12 → Elements
2. Найдите кнопку "Добавить урок"
3. Проверьте computed styles:
   - `z-index` должен быть `101`
   - `pointer-events` должен быть `auto`

### Проблема 4: ДИАЛОГ НЕ ПОЯВЛЯЕТСЯ

**Проверьте:**
```javascript
// В консоли после клика:
const dialogs = document.querySelectorAll('[role="dialog"]');
console.log('Dialogs:', dialogs);
```

Если диалог ЕСТЬ в DOM, но не виден → проблема с z-index или opacity.

**Решение:**
Проверьте в DevTools:
1. Найдите элемент с `role="dialog"`
2. Computed styles:
   - `z-index` должен быть `10001`
   - `opacity` должен быть `1`
   - `display` должен быть `grid` или `block`
