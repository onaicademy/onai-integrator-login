## 📊 ТЕКУЩИЙ ФУНКЦИОНАЛ (ЧТО УЖЕ ЕСТЬ)

### 1. **Основные Метрики** ✅
- **Revenue (Доход)** - из AmoCRM, в тенге
- **Spend (Затраты)** - Facebook Ads, в долларах
- **ROAS** - Return on Ad Spend (ключевая метрика)
- **CPA** - Cost Per Acquisition (стоимость продажи)
- **Sales** - количество продаж
- **Clicks** - клики по рекламе
- **Impressions** - показы рекламы
- **CTR** - Click-Through Rate (кликабельность)
- **Reach** - охват аудитории
- **Leads** - количество лидов

### 2. **Видео Метрики** 🎬
- **Plays** - просмотры видео
- **Thruplay** - досмотр до 15 сек
- **Completions** - досмотр до конца
- **Completion Rate** - процент досмотров
- **Avg Watch Time** - среднее время просмотра
- **Retention** - удержание аудитории (25%, 50%, 75%, 100%)
- **Top Video Creatives** - топ-3 креатива по вовлеченности

### 3. **Геймификация и Рейтинг** 🏆
- **4-уровневая система рангов:**
  - 🏆 #1 - "ЛЕГЕНДАРНО" (зелёный shimmer эффект)
  - 🥈 #2 - "ЭПИЧНО"
  - 🥉 #3 - "ХОРОШО"
  - ⭐ #4 - "ТРЕБУЕТ ДОРАБОТОК"
- Ранжирование по ROAS
- Визуальные медали и цветовая дифференциация

### 4. **AI Рекомендации** 🤖
- Groq AI (Llama 3.3 70B)
- Персонализированные рекомендации для каждой команды
- Анализ метрик: ROAS, CPA, CTR, видео-вовлечённость
- Короткие actionable советы

### 5. **Топ Кампании** 📈
- **ТОП по продажам** (UTM из AmoCRM)
- **ТОП по CTR** (кликабельность)
- **ТОП по видео-вовлечённости** (досмотры)

### 6. **UX/UI Фичи** 🎨
- Tripwire brand colors (#00FF88 - зелёный)
- Тёмная тема (минимализм)
- Responsive design (mobile, tablet, desktop)
- Интерактивные tooltips для метрик
- USD/KZT переключатель валют
- Периоды: 7д, 14д, 30д, custom date
- Фильтр по командам
- Auto-refresh каждые 10 минут
- Manual sync кнопка

### 7. **Интеграции** 🔌
- Facebook Ads API (permanent token ♾️)
- AmoCRM API (sales tracking)
- Groq AI (recommendations)
- Supabase (database)
- Real-time currency exchange rate (USD-KZT)

## 🚀 ПРЕДЛОЖЕНИЯ ДЛЯ УЛУЧШЕНИЯ МОТИВАЦИИ

### 🎯 **1. ПЕРСОНАЛЬНЫЕ KPI ЦЕЛИ**

**Проблема:**  
Таргетологи видят текущие цифры, но не понимают **"Насколько хорошо я работаю? Куда мне стремиться?"**

**Решение:**
- Установить **месячные цели** для каждой команды:
  - Цель по ROAS (например, 2.0x)
  - Цель по Sales (например, 50 продаж/месяц)
  - Цель по Revenue (например, ₸250,000)
  - Цель по CPA (например, < $50)

**Визуализация:**
```
┌─────────────────────────────────────────────┐
│ 🎯 ТВОИ ЦЕЛИ НА ДЕКАБРЬ                    │
├─────────────────────────────────────────────┤
│ Revenue:  ₸175K / ₸250K  [━━━━━━━━░░] 70%  │
│ Sales:    35 / 50        [━━━━━━░░░░] 70%  │
│ ROAS:     1.5x / 2.0x    [━━━━━░░░░░] 75%  │
│ CPA:      $48 / $50      [━━━━━━━━━░] 96% ✅│
└─────────────────────────────────────────────┘

💡 Ещё 15 продаж до цели! Осталось 13 дней.
```

**Мотивация:**
- ✅ Прогресс-бары показывают путь к успеху
- ✅ Ежедневная цель (разбить на дни)
- ✅ Сравнение с прошлым месяцем

### 📈 **2. СРАВНЕНИЕ С ПРОШЛЫМ ПЕРИОДОМ**

**Проблема:**  
Нет контекста - "Я расту или падаю?"

**Решение:**
- Добавить **стрелки роста/падения** для каждой метрики
- Показать % изменения относительно предыдущего периода

**Визуализация:**
```
┌──────────────────────────┐
│ Revenue                  │
│ ₸35,000  ↑ +25%         │  ← Зелёная стрелка вверх
│ vs прошлая неделя        │
└──────────────────────────┘

┌──────────────────────────┐
│ CPA                      │
│ $48  ↓ -15%             │  ← Зелёная стрелка вниз (хорошо)
│ vs прошлая неделя        │
└──────────────────────────┘
```

**Данные:**
- Сравнение 7д vs предыдущие 7д
- Сравнение текущий месяц vs прошлый месяц
- Цветовая индикация: рост Revenue = 🟢, падение CPA = 🟢

### 🔮 **3. ПРОГНОЗ НА КОНЕЦ МЕСЯЦА**

**Проблема:**  
"Успею ли я достичь месячных целей?"

**Решение:**
- AI прогнозирует финальные цифры на основе текущего темпа
- Показывает вероятность достижения целей

**Визуализация:**
```
┌─────────────────────────────────────────────┐
│ 🔮 ПРОГНОЗ НА КОНЕЦ ДЕКАБРЯ                │
├─────────────────────────────────────────────┤
│ Revenue:  ~₸235K  (94% от цели)            │
│ Sales:    ~47     (94% от цели)            │
│                                             │
│ ⚠️  Нужно +3 продажи/неделю для цели!      │
│ 💡 Увеличь бюджет на 15% или улучши CTR    │
└─────────────────────────────────────────────┘
```

**Мотивация:**
- Показывает **gap to goal** (разрыв до цели)
- Даёт конкретные советы, что сделать
- Создаёт срочность ("осталось 10 дней!")

### 🎮 **4. ГЕЙМИФИКАЦИЯ: УРОВНИ И ДОСТИЖЕНИЯ**

**Проблема:**  
Монотонность - одни и те же метрики каждый день

**Решение:**
- **Уровни таргетолога:**
  - Новичок (Level 1-5)
  - Специалист (Level 6-10)
  - Эксперт (Level 11-15)
  - Мастер (Level 16-20)
  - Легенда (Level 21+)

- **Достижения (Badges):**
  - 🏆 "Первые 10 продаж"
  - 💎 "ROAS 3.0x+"
  - 🔥 "7 дней подряд с продажами"
  - 🎯 "CTR выше 2%"
  - 💰 "CPA ниже $40"
  - 🚀 "100K revenue за неделю"
  - 🎬 "Видео с 20% досмотром"

**Визуализация:**
```
┌─────────────────────────────────────────────┐
│ Kenesary 👑                Level 12 (Эксперт)│
├─────────────────────────────────────────────┤
│ XP: 8,450 / 10,000  [━━━━━━━━░░] 84%       │
│                                              │
│ Unlocked Badges:                             │
│ 🏆 10 продаж  💎 ROAS 3x  🔥 7-day streak   │
│                                              │
│ Next Badge: 💰 CPA Мастер (45/50 продаж)    │
└─────────────────────────────────────────────┘
```

**XP (Опыт) начисляется за:**
- +100 XP - каждая продажа
- +500 XP - ROAS выше 2.0x в день
- +200 XP - CTR выше 1.5%
- +50 XP - каждый $100 revenue

**Мотивация:**
- Соревнование между командами за уровень
- Публичный лидерборд
- Награды за достижения

## 📊 ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ

### **Новые API Endpoints:**
```
GET  /api/traffic/goals/:team                 - Цели команды
GET  /api/traffic/comparison/:team/:period    - Сравнение периодов
GET  /api/traffic/forecast/:team              - Прогноз на месяц
GET  /api/traffic/heatmap/:team/:metric       - Тепловая карта
GET  /api/traffic/creatives/:team/top         - Топ креативы с thumbnails
GET  /api/traffic/audience/:team              - Демография аудитории
GET  /api/traffic/timeline/:team              - История событий
POST /api/traffic/goals/:team                 - Установить цели
GET  /api/gamification/profile/:team          - Уровень, XP, badges
GET  /api/gamification/leaderboard            - Лидерборд
```

### **Новые Supabase Таблицы:**
```sql
-- Цели команд
CREATE TABLE traffic_goals (
  team TEXT PRIMARY KEY,
  roas_target DECIMAL,
  sales_target INTEGER,
  revenue_target DECIMAL,
  cpa_target DECIMAL,
  period TEXT, -- 'monthly', 'weekly'
  created_at TIMESTAMP DEFAULT NOW()
);

-- Геймификация
CREATE TABLE gamification_profiles (
  team TEXT PRIMARY KEY,
  level INTEGER DEFAULT 1,
  xp INTEGER DEFAULT 0,
  badges JSONB DEFAULT '[]',
  created_at TIMESTAMP DEFAULT NOW()
);

-- История событий
CREATE TABLE traffic_timeline (
```
