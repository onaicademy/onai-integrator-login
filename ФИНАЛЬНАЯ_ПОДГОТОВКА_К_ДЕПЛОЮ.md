## 🆘 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ:

### **Проблема: Сайт не загружается**
```bash
# 1. Проверь статус сервиса:
pm2 status

# 2. Перезапусти:
pm2 restart onai-platform

# 3. Посмотри логи:
pm2 logs onai-platform
```

### **Проблема: База данных не работает**
```
1. Supabase Dashboard → Settings → API
2. Проверь что API keys правильные в .env
3. Проверь что Supabase URL правильный
```

### **Проблема: Telegram боты не отправляют**
```
1. Проверь токены в .env:
   VITE_AI_MENTOR_BOT_TOKEN=8380600260:...
   VITE_AI_ANALYST_BOT_TOKEN=8400927507:...

2. Проверь что Chat ID правильный

3. Проверь логи в консоли браузера (F12)
```
