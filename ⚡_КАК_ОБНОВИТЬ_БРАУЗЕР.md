# ⚡ КАК ОБНОВИТЬ БРАУЗЕР

## Изменения сохранены, но не видны?

Попробуйте эти шаги:

### 1. **Hard Refresh** (жесткое обновление)

Windows:
```
Ctrl + Shift + R
или
Ctrl + F5
```

Mac:
```
Cmd + Shift + R
```

### 2. **Очистить кеш и перезагрузить**

Chrome/Edge:
1. Откройте DevTools (F12)
2. Правый клик на кнопке обновления ↻
3. Выберите "Очистить кеш и жесткое обновление"

### 3. **Если не помогло - перезапустите dev сервер**

Остановите сервер:
```
Ctrl + C
```

Запустите заново:
```
npm run dev
```

### 4. **Проверьте что открыта правильная страница**

```
http://localhost:8080/aimarathon
```

## ✅ Что должно быть видно

После обновления логотип должен быть **БЕЛЫМ**:
- Буквы "O" и "N" (toggle) - белые
- Буквы "AI" - белые
- Текст "DIGITAL ACADEMY" - белый
- Разделитель - белый

Через 2 секунды toggle становится **ЗЕЛЁНЫМ** (#B1FF32) ✅
