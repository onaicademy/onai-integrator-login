## ✅ ЧТО БЫЛО ИСПРАВЛЕНО

### 1. **Главная проблема: API key не передавался в headers**

**Файл:** `src/lib/supabase.ts`

**Было:**
```typescript
export const supabase = createClient(supabaseUrl, supabaseKey, {
  auth: {
    storage: localStorage,
    persistSession: true,
    autoRefreshToken: true,
  }
})
```

**Стало:**
```typescript
export const supabase = createClient(supabaseUrl, supabaseKey, {
  auth: {
    storage: localStorage,
    persistSession: true,
    autoRefreshToken: true,
  },
  global: {
    headers: {
      'apikey': supabaseKey,
      'Authorization': `Bearer ${supabaseKey}`
    }
  }
})
```

**Почему это важно:**
- Supabase требует API key в каждом запросе
- Теперь `apikey` и `Authorization` headers передаются явно
- Это решает ошибку "No API key found in request"
