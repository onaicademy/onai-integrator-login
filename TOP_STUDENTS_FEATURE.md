# Функция "Топ Ученики" на странице Admin/Activity

## Описание
На странице `/admin/activity` добавлена функциональность для просмотра топ-15 учеников платформы.

## Реализованные возможности

### 1. Навигационная панель
- Добавлена карточка с навигационными вкладками сразу после заголовка страницы
- Вкладки:
  - **Обзор** - основной вид страницы
  - **Топ Ученики** ⭐ - открывает модальное окно с лучшими учениками
  - **Все Ученики** - зарезервировано для будущего функционала

### 2. Модальное окно "Топ 15 Учеников"
При клике на кнопку "Топ Ученики" открывается современное модальное окно с:

#### Особенности дизайна:
- ✅ Плавная анимация появления (`animate-in fade-in-0 zoom-in-95 duration-300`)
- ✅ Встроенный крестик для закрытия (компонент Dialog от shadcn/ui)
- ✅ Адаптивная ширина (`max-w-6xl`)
- ✅ Скроллинг при переполнении контента
- ✅ Соответствие текущей стилистике платформы

#### Содержание:
**Таблица с детальными данными о 15 учениках:**
- Позиция в рейтинге (с цветными бейджами для топ-3)
- Имя и аватар ученика
- **XP** - опыт (с иконкой молнии)
- **Стрик** - дни без пропусков (с иконкой огня)
- **Уроков завершено** (с иконкой книги)
- **Вопросов куратору** (с иконкой сообщений)
- **Продуктивность** - % (с цветовой кодировкой)
- **Оценка** - рейтинг от 0 до 5
- **Поведение** - качественная оценка
- **Последняя активность**

**Статистика в футере:**
- Средняя продуктивность топ-15
- Суммарный XP всех топ-учеников
- Всего завершённых уроков
- Средний стрик
- Всего вопросов куратору

#### UI/UX улучшения:
- Hover-эффекты на строках таблицы
- Цветовая кодировка по важности метрик
- Анимация масштабирования бейджей при наведении
- Иконки в заголовках столбцов для лучшей читаемости
- Градиентные фоны для статистических карточек

### 3. Фейковые данные (Mock Data)
Список из 15 учеников с разнообразными показателями:
1. **Иван** - 2000 XP, 9 дней стрик, 58 уроков
2. **Алиса** - 1850 XP, 10 дней стрик, 52 урока
3-15. Дополнительные ученики с убывающими показателями

Данные рассчитаны на основе совокупности:
- XP (опыт)
- Завершённые уроки
- Вовлечённость (продуктивность, вопросы)
- Стрик (отсутствие пропусков)

## Как использовать

1. Откройте страницу `/admin/activity`
2. Найдите навигационную панель под заголовком
3. Нажмите кнопку "Топ Ученики"
4. Откроется модальное окно с таблицей топ-15 учеников
5. Закройте окно крестиком в правом верхнем углу или кликом вне окна
