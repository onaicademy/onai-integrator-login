## ✅ Отключен автодеплой

### Что изменилось:

**До:**
```yaml
on:
  push:
    branches:
      - main
```
➡️ При каждом `git push` автоматически деплоилось на сервер

**После:**
```yaml
on:
  workflow_dispatch:  # Только вручную
```
➡️ Деплой происходит **только когда ты сам запускаешь**

## 🔄 Как работает ручной деплой

### Схема работы:

```
1. Пишешь код локально
   ↓
2. git add . && git commit -m "описание"
   ↓
3. git push origin main  (отправляет на GitHub)
   ↓
4. НА ЭТОМ АВТОДЕПЛОЙ НЕ ПРОИСХОДИТ ❌
   ↓
5. Ты сам запускаешь деплой когда готов:
   - Вариант A: ./manual-deploy.sh
   - Вариант B: GitHub Actions UI
```

## 🎯 Использование скрипта

### Скрипт: `manual-deploy.sh`

Этот скрипт:
1. ✅ Подключается к серверу
2. ✅ Забирает последние изменения с GitHub
3. ✅ Устанавливает зависимости
4. ✅ Собирает production build
5. ✅ Перезапускает Nginx

### Как использовать:

```bash
# Запуск деплоя
./manual-deploy.sh

# Или с явным указанием bash
bash manual-deploy.sh
```

**Скрипт спросит подтверждение перед деплоем:**
```
❓ Продолжить деплой? (y/n):
```

Нажми `y` чтобы продолжить, или `n` чтобы отменить.

```bash

./manual-deploy.sh
```

## 🌐 Запуск через GitHub UI

Если не хочешь использовать скрипт, можешь запускать деплой через GitHub:

### Шаги:

1. Открой https://github.com/onaicademy/onai-integrator-login/actions

2. Выбери **"Deploy to DigitalOcean Droplet"** в списке слева

3. Нажми **"Run workflow"** справа

4. Выбери ветку **main**

5. Нажми **"Run workflow"** (зелёная кнопка)

6. Дождись завершения (обычно ~1-2 минуты)

## 🛠️ Дополнительные команды

### Просмотр истории коммитов:

```bash
# Краткий список
git log --oneline

# Последние 10 коммитов
git log --oneline -10

# С датами и авторами
git log --pretty=format:"%h - %an, %ar : %s"
```

### Откат изменений (если нужно):

```bash
# Откатить последний коммит (не удаляя изменения)
git reset --soft HEAD~1

# Откатить изменения в файле
git checkout -- src/pages/Profile.tsx

# Посмотреть что изменилось
git diff
```

## 🚨 Troubleshooting

### Проблема: "Permission denied" при запуске скрипта

```bash
chmod +x manual-deploy.sh
```

### Проблема: "SSH connection failed"

Проверь что можешь подключиться:
```bash
ssh root@178.128.203.40
```

### Проблема: "Git pull failed"

На сервере могут быть незакоммиченные изменения:
```bash
ssh root@178.128.203.40
cd /var/www/onai-integrator-login
git status
git stash  # Сохранить изменения
git pull origin main
```

### Проблема: "Build failed"

Проверь логи на сервере:
```bash
ssh root@178.128.203.40
cd /var/www/onai-integrator-login
npm run build
```
