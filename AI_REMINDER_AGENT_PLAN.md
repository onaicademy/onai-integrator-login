# 🤖 AI АГЕНТ-НАПОМИНАТЕЛЬ В TELEGRAM - ПЛАН

## 📋 ОБЗОР

**Цель:** Создать отдельного AI агента в Telegram, который будет напоминать студентам о задачах из канбан-доски и мотивировать их на выполнение.

---

## 🎯 ФУНКЦИОНАЛ

### 1️⃣ Напоминания о задачах
- ⏰ **За 1 час до дедлайна:** "У тебя остался 1 час до выполнения задачи '[Название]'. Обязательно выполни!"
- 📅 **Утренний дайджест:** Список задач на сегодня (в 9:00)
- 🌙 **Вечерний отчет:** Что выполнено сегодня + мотивация на завтра (в 21:00)

### 2️⃣ Мотивационные сообщения
- 🔥 При длинном периоде без активности: "Эй, давно не видел тебя в деле! Есть 3 задачи, которые ждут..."
- 🎉 При выполнении задачи: "Отлично! +20 XP! Так держать!"
- 💪 При streak: "7 дней подряд выполняешь задачи! Ты машина!"

### 3️⃣ Управление через команды
```
/tasks_on - Включить напоминания
/tasks_off - Отключить напоминания
/tasks_today - Показать задачи на сегодня
/tasks_add [название] - Добавить задачу
/tasks_complete [ID] - Отметить как выполненную
/tasks_stats - Статистика по задачам
```

---

## 🏗️ АРХИТЕКТУРА

### Компоненты:
1. **Telegram Bot** (Node.js + Telegraf)
2. **Supabase Database** (user_goals таблица)
3. **Cron Jobs** (для планирования напоминаний)
4. **OpenAI API** (для генерации мотивационных сообщений)

### Новая таблица: `user_notifications_settings`
```sql
CREATE TABLE user_notifications_settings (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  user_id UUID NOT NULL REFERENCES auth.users(id) ON DELETE CASCADE,
  telegram_chat_id BIGINT,
  telegram_username TEXT,
  notifications_enabled BOOLEAN DEFAULT TRUE,
  reminder_1h_before BOOLEAN DEFAULT TRUE,
  morning_digest BOOLEAN DEFAULT TRUE,
  evening_report BOOLEAN DEFAULT TRUE,
  motivation_messages BOOLEAN DEFAULT TRUE,
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW(),
  UNIQUE(user_id)
);
```

## 🔄 FLOW ДИАГРАММА

```
1. Студент создает задачу на платформе
        ↓
2. Задача сохраняется в user_goals (с due_date)
        ↓
3. Cron Job каждые 10 минут проверяет:
   - Есть ли задачи с дедлайном через 1 час?
   - Есть ли пользователи с включенными напоминаниями?
        ↓
4. Если ДА → Отправляет Telegram сообщение:
   "⏰ [Имя], у тебя остался 1 час до задачи '[Название]'!
    💡 Совет: разбей на маленькие шаги и начни прямо сейчас!"
        ↓
5. Студент нажимает кнопку "Выполнено" в Telegram
        ↓
6. Бот обновляет статус задачи → API → user_goals
        ↓
7. Триггер начисляет XP (+20)
        ↓
8. Бот отправляет: "🎉 Отлично! +20 XP. Так держать!"
```

## 🛠️ ТЕХНИЧЕСКИЙ СТЕК

### Backend (Telegram Bot):
```
- Node.js + TypeScript
- Telegraf (Telegram Bot Framework)
- node-cron (для планирования)
- Supabase Client
- OpenAI API (для мотивационных сообщений)
```

### Структура проекта:
```
backend/
  telegram-bot/
    src/
      index.ts          # Запуск бота
      handlers/
        commands.ts     # /tasks_on, /tasks_off и т.д.
        callbacks.ts    # Кнопки "Выполнено", "Отложить"
      schedulers/
        reminders.ts    # Cron job для напоминаний
        digest.ts       # Утренний/вечерний дайджест
      services/
        goalsService.ts # Работа с user_goals
        aiService.ts    # Генерация мотивационных сообщений
      utils/
        messages.ts     # Шаблоны сообщений
```
