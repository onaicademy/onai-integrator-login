### **ШАГ 7: НАСТРОЙКА NGINX** 🌐

**7.1. Проверь конфигурацию Nginx:**

```bash
nano /etc/nginx/sites-available/integratoronai.kz
```

**Должно быть примерно так:**

```nginx
server {
    listen 80;
    server_name integratoronai.kz www.integratoronai.kz;
    
    # Redirect HTTP to HTTPS
    return 301 https://$server_name$request_uri;
}

server {
    listen 443 ssl http2;
    server_name integratoronai.kz www.integratoronai.kz;
    
    # SSL сертификаты (Let's Encrypt)
    ssl_certificate /etc/letsencrypt/live/integratoronai.kz/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/integratoronai.kz/privkey.pem;
    
    # Root директория
    root /var/www/onai-integrator-login/dist;
    index index.html;
    
    # Gzip сжатие
    gzip on;
    gzip_types text/plain text/css application/json application/javascript text/xml application/xml application/xml+rss text/javascript;
    
    location / {
        try_files $uri $uri/ /index.html;
    }
    
    # Кэширование статики
    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg|woff|woff2|ttf|eot)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }
}
```

**7.2. Проверь и перезагрузи Nginx:**

```bash
# Проверь синтаксис
nginx -t

# Если OK, перезагрузи
systemctl reload nginx

# Проверь статус
systemctl status nginx
```
